## Run Locally

Clone the project

```bash
  git clone https://github.com/Mamun1646/CRUD-Operation-With-Rest-API.git
```

Go to the project directory

```bash
  cd CRUD-Operation-With-Rest-API
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon index.js
```
